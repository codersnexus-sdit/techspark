##  Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/shondsouza/Techspark.git
   cd Techspark
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the application** *
   ```bash
   npm run dev
   ```

4. **Access the platform**
   - Main Application: http://localhost:3000
